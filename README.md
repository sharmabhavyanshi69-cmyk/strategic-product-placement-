# Strategic Product Placement Analysis

An interactive Business Intelligence application that analyzes product placement trends across movies and TV shows using Tableau, Python (Flask), and data visualization techniques. The project helps identify placement effectiveness, brand visibility, and audience engagement through interactive dashboards and stories.

# 📌 Project Overview

The **Strategic Product Placement Analysis** project aims to analyze how brands strategically place products in entertainment media to maximize visibility, audience engagement, and marketing impact.

Using **Tableau Public** dashboards integrated into a **Flask (Python)** web application, the system provides interactive visualizations that help users explore placement patterns, brand performance, industry trends, and key business insights.

The application enables marketing professionals, business analysts, researchers, and students to make informed, data-driven decisions by understanding product placement strategies across different media platforms.

# 🚀 Installation

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/Strategic-Product-Placement-Analysis.git
```

## 2. Navigate to the Project Folder

```bash
cd Strategic-Product-Placement-Analysis
```

## 3. Create a Virtual Environment

```bash
python -m venv venv
```

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

## 4. Install Dependencies

```bash
pip install flask
```

or

```bash
pip install -r requirements.txt
```

## 5. Run the Application

```bash
python app.py
```

## 6. Open in Browser

```text
http://127.0.0.1:5000/
```
