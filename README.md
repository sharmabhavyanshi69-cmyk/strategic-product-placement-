Strategic Product Placement Analysis

An interactive Business Intelligence application that analyzes product placement trends across movies and TV shows using Tableau, Python (Flask), and data visualization techniques. The project helps identify placement effectiveness, brand visibility, and audience engagement through interactive dashboards and stories.

📌 Project Overview

The Strategic Product Placement Analysis project aims to analyze how brands strategically place products in entertainment media to maximize visibility and marketing impact.

Using Tableau dashboards integrated into a Flask web application, the system provides interactive visualizations that help users explore placement patterns, brand performance, industry trends, and key insights.

The project enables marketing professionals, researchers, and students to make data-driven decisions by understanding product placement strategies.
